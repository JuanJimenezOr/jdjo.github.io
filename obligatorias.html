Inteligencia artificial

La inteligencia artificial siempre ha sido un tema muy interesante la IA empezó desde el año 1950 y su base se ha mantenido desde entonces, normalmente en la IA una gran cantidad de datos por lo cual para resolver un problema pueden existir vatios caminos que lleguenn a diferentes resultados. Nils Jhon la divide en dos pilares fundamentales que son los algoritmos genéticos y las redes neuronales. La diferencia entre estos dos es que los algoritmos genéticos están basados en la evolución mientras que las redes neuronales funcionan en base a resultados ya obtenidos. Algo interesante es sobre el sistema inteligente desarrollado por IBM llamado Watson que es capaz de interactuar con las personas

Tendencias en el desarrollo web

Esta conferencia es muy interesante ya que se habla sobre las herramientas actuales que se están utilizando para el desarrollo web, este se divide en 2 partes:
-Front-end que es todo lo que ve el cliente y trabaja con tecnologías como html,css y js
-Back-end- donde se encuentra el servidor y se utiliza para cosas como lógica de negocios, bases de datos, Web Services, etc.
También se hablo de las mejores que se han tenido en las aplicaciones web gracias a las Progressive Web Aplication y Single web aplication. También dio 10 tips muy importantes para nosotros los egresado:
*Elegir un lenguaje de programación y hacerlo tu primer amor
*Una vez que te sientas cómodo prueba otro lenguaje
*Saber como se comportan las estructuras de datos
*Entrenar habilidad de programar algoritmos
*Entender como funciona por detrás un framework
*Conocer manejadores de versiones como git
*No esperarse hasta 9no para generar experiencia
*Trata de crear tu propio negocios
*Leer mucho
*Hablar inglés

Herramientas y técnicas para incrementar la calidad del código

Esta conferencia me gusto porque se nos dieron consejos para mejorar el software que desarrollemos y los 4 puntos para mantener un buen desarrollo son:
1- Mantener a los desarrolladores felices
2- Mantener a los desarrolladores relajados
3- Contratar al mejor talento de tu localidad
4- Visitar conferencias

Además se menciono que para que un software sea de calidad debe cumplir con estos 8 puntos:
1-Debe cumplir su propósito
2-Informa adecuadamente cuando un problema ocurre
3-Tiene pruebas
4-Se entiende su intención con solo leerlo
5-Sigue un estilo desde el inicio
6-Utiliza flujo de colaboración adecuado
7-Implementa correctamente los patrones de diseño
8- Se puede utilizar intuitivamente

Puntos para una colaboración efectiva usando control de versiones:
1- Definir una rama estable para la producción
2- Definir una rama estable para el ambiente de investigación
3- Definir el estándar para nombrar las ramas de trabajo individuales
4- Definir estándar para los commit messages
5- Definir estándar para la revisión del código
todos estos puntos son de gran ayuda para empezar a mejorar y tener siempre en mente estos consejos al momento de desarrollar

Eduardo Figerola Mota

Hacia dónde va la inv. en las ciencias de la información

Esta conferencia nos da una idea de cómo están los posgrados en México, actualmente hay 2155 programas de postgrado de calidad donde el área de ingeniería es la que tiene más estudios de postgrado, de esa cantidad en Nayarit solo hay 12 y ninguno en el área de ingeniería. Esto nos demuestra que a Nayarit le faltan maestros calificados para llevar a cabo estas maestrías, ya que la conferencista comparo los números de Nayarit con los estados vecinos y es mucho mayor los programas de posgrado que tiene.
Esta conferencia nos sirve para conocer los requisaos si uno quiere estudiar alguna maestría al momento de salir, para eso es necesario contar con lo siguiente:
*Promedio de 8
*Tener compromiso de estar solo ahí
*Dominio del idioma inglés
*Presentar examen ceneval
